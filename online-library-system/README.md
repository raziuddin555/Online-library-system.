# Online Library System

A React-based Online Library System that allows users to browse books, search books, view detailed information, and add new books using Redux Toolkit for state management.

---

## Features

- Browse books by category
- Search books by title or author
- View detailed information about each book
- Add new books dynamically
- Dynamic routing using React Router
- State management using Redux Toolkit
- 404 Not Found page for invalid routes
- Responsive and clean UI

---


## Installation & Setup

### 1. Clone the repository

```bash
git https://github.com/raziuddin555/Online-library-system./tree/main/online-library-system 
```

### 2. Open the project folder

```bash
cd online-library-system
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm run dev
```

### 5. Open in browser

```bash
http://localhost:5173/
```
